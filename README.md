## Overview

ShopO is a modern front-end e-commerce application designed to deliver a clean, responsive, and intuitive shopping experience.

The project demonstrates real-world React development practices including reusable component architecture, global state management using Context API, client-side routing, dynamic product rendering, shopping cart functionality, filtering, sorting, responsive layouts, and modern UI development.

The application follows component-driven development principles and emphasizes scalability, maintainability, and clean code organization.

## Installation

Clone the repository

```bash
git clone https://github.com/Sawera-kiran/shopo-web.git
```

Navigate into the project

```bash
cd shopo-web
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
